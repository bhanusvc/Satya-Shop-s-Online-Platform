# Satya-Shop-s-Online-Platform
Single Shop Online Platform to sell goods.
Main Goal is to create an Online Platform for Small Shops to sell their their Products using this website.
The UI must be Simple and Easy to Navigate along with necessary Functionalities.
**Goal/Structure** as of **01-01-2026**: 
  **1.** Catalog, Cart, Payment Page with other niche pages of a website. For Payment Considering UPI for now. 
  **2.** Should have a Messaging System such as Order Confirmation(to Mail).
